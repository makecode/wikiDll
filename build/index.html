Programs that open MOV files

A file extension is three or four characters at the end of a filename, such as .DOCX, .PDF or .TXT. File extensions tell you the file type, and which programs can open it. These extensions are used in Microsoft Windows, Mac OS X and Linux Operating Systems.

What is a file extension?

A file extension is the characters after the last dot in a file name. For example, in the file name "winmail.dat", the file extension is "dat". It helps Windows select the right program to open the file.
